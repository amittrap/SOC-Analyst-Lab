# SOC Analyst Lab

## Objective
This project demonstrates practical SOC Analyst skills including log analysis,
alert triage, detection engineering, and incident reporting.

## Skills Covered
- Linux & Windows log analysis
- Alert investigation and triage
- Sigma rule creation
- Incident response documentation

## Tools & Technologies
- Linux auth.log
- Windows Event Logs
- Sigma
- MITRE ATT&CK
- SIEM concepts

## Scenarios
- SSH Brute-force attack detection
- Windows authentication failures
- Suspicious PowerShell execution

## Outcome
Improved ability to detect, analyze, and respond to security incidents
in a SOC environment.
